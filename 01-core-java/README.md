## 📖 Overview

This section covers the foundational and advanced Core Java topics that are **consistently tested** in backend engineering interviews at top tech companies. Mastery of these topics is essential before moving to frameworks, databases, or system design.

| Multithreading | Thread lifecycle, locks, executors, concurrency utilities |
| Networking | Sockets, HTTP, REST, java.net package |
| Exception Handling | Checked/unchecked, try-with-resources, anti-patterns |
| Collections | HashMap internals, ConcurrentHashMap, fail-fast vs fail-safe |
| Strings & Regex | String pool, immutability, Pattern/Matcher |
| Advanced Java | Generics, enums, records, sealed classes, JPMS |

## 🎯 Interview Strategy

### Tier 1 (Must Know — Asked in 90%+ of interviews)
- HashMap internals, `equals()` and `hashCode()` contract
- Thread lifecycle, `synchronized` vs `volatile`
- Checked vs unchecked exceptions
- String immutability and String pool
- SOLID principles with examples

### Tier 2 (Should Know — Asked in 60–80% of senior interviews)
- `ConcurrentHashMap` internals
- `CompletableFuture` and async programming
- Generics and type erasure
- Fail-fast vs fail-safe iterators
- `ForkJoinPool` and work-stealing

### Tier 3 (Differentiators — Asked in staff/principal interviews)
- JVM memory model (JMM) happens-before guarantees
- `StampedLock` vs `ReentrantReadWriteLock`
- Records, sealed classes, pattern matching (Java 16+)
- JPMS (Java Platform Module System)
